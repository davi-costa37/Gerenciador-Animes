Anime Manager
Um gerenciador de animes moderno e prático para organizar sua lista de episódios assistidos, favoritar suas obras preferidas e acompanhar o status de cada exibição (assistindo, concluído, planejado para assistir).

🚀 Funcionalidades
Cadastro de Animes: Adicione novos animes com título e qauntidade de episódios.

Controle de Episódios: Marque em qual episódio você parou para nunca se perder.

Status de Visualização: Organize seus animes por categorias (Assistindo, Concluído, Pausado, Quero Assistir).


Filtros Inteligentes: Busque animes por nome, gênero ou status de progresso.

🛠️ Tecnologias Utilizadas
O projeto foi desenvolvido utilizando as seguintes tecnologias:

Frontend: HTML5, CSS3, JavaScript (ES6+) and Firebase

Hospedagem/Deploy: Vercel
